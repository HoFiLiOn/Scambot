# 🤖 Scam Bot - Бот для проверки мошенников

## 📋 Описание
Бот для проверки пользователей по базе мошенников. Позволяет искать по @username и по ID.

## 🚀 Команды

### Для всех:
- `/start` - главное меню
- `/help` - помощь
- `/search @username` - проверить по юзернейму
- `/search 123456789` - проверить по ID

### Для админов:
- `/addscam @username` - добавить в скам-базу
- `/removescam @username` - удалить из скам-базы
- `/scamlist` - список всех мошенников

## 👮 Админ-панель
В админ-панели доступны:
- 📊 Статистика
- 👥 Список пользователей
- 🔍 Поиск пользователей
- 📋 Скам-список с пагинацией
- ➕ Добавление в скам
- ➖ Удаление из скам
- 🔎 Поиск по скам-базе
- 📨 Рассылка
- 📜 Логи админов
- 📈 Моя статистика

## 🛠 Установка на BotHost
1. Загрузи все файлы на GitHub
2. На BotHost укажи репозиторий
3. Команда запуска: `python bot.py`
4. Готово!
